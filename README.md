# МояКойка — Аренда общежитий

SPA-приложение для поиска и размещения объявлений об общежитиях в Калининградской области.

---

## Как разместить сайт бесплатно (3 способа)

### Способ 1 — GitHub Pages (рекомендуется, HTTPS бесплатно)

1. Создайте репозиторий на [github.com](https://github.com).
2. Загрузите все файлы проекта в репозиторий (в корень).
3. Зайдите в **Settings → Pages**.
4. В разделе **Source** выберите ветку `main` и папку `/ (root)`.
5. Нажмите **Save**. Через 1-2 минуты сайт будет доступен по адресу:
   ```
   https://ВАШ_НИК.github.io/ИМЯ_РЕПОЗИТОРИЯ/
   ```

> Если у вас есть свой домен — в настройках Pages введите его в поле **Custom domain**, а в DNS добавьте CNAME-запись на `ВАШ_НИК.github.io`.

---

### Способ 2 — Netlify (drag-and-drop, HTTPS бесплатно)

1. Зайдите на [netlify.com](https://netlify.com) и авторизуйтесь (можно через GitHub).
2. Перетащите папку с проектом в область загрузки на главной странице.
3. Через несколько секунд сайт будет доступен по адресу вида:
   ```
   https://название-xxx.netlify.app
   ```
4. В **Site settings → Domain management** можно подключить свой домен или сменить поддомен.

---

### Способ 3 — Vercel

1. Зайдите на [vercel.com](https://vercel.com) и авторизуйтесь через GitHub.
2. Импортируйте репозиторий с проектом.
3. Сайт автоматически задеплоится с HTTPS.

## Как подключить вход через ВКонтакте

### Шаг 1 — Создайте приложение VK

1. Зайдите на [vk.com/dev](https://vk.com/dev).
2. Нажмите **Создать приложение**.
3. Укажите:
   - **Название**: «МояКойка» (или любое)
   - **Тип**: «Веб-сайт»
   - **Базовый домен**: ваш домен (например, `моякойка.рф` или `xxx.github.io`)
   - **Доверенный redirect URL**: `https://ВАШ_ДОМЕН/` (точно так же, как будет открываться сайт, со слэшем на конце)
4. Сохраните. Откройте раздел **Настройки** → скопируйте **ID приложения**.

### Шаг 2 — Вставьте ID в код

Откройте файл `auth.js`, найдите строку:
```javascript
const VK_APP_ID = '';
```

Вставьте ваш ID:
```javascript
const VK_APP_ID = '12345678';
```

Сохраните файл и загрузите обновлённую версию на хостинг.

### Шаг 3 — Проверьте

1. Откройте сайт по **точному тому же адресу**, который указан в redirect URL в VK.
2. Нажмите **«Войти через ВКонтакте»** — должен открываться VK OAuth.
3. После авторизации VK перенаправит обратно на сайт с токеном, и вход произойдёт автоматически.

> **Важно:** Redirect URL в настройках VK и адрес сайта в браузере должны совпадать **символ в символ**. Если в VK указано `https://xxx.github.io/`, а вы открываете `http://xxx.github.io/` (без HTTPS) или `https://xxx.github.io/moyakoyka/` (с подпапкой) — VK выдаст ошибку.

## Структура проекта

```
├── index.html        # Главная страница (SPA)
├── style.css         # Стили
├── app.js            # Основная логика приложения
├── auth.js           # Авторизация, регистрация, VK OAuth
├── data.js           # Демо-данные, утилиты, localStorage
├── router.js         # Модуль роутера
├── server.js         # Локальный сервер Node.js (для разработки)
└── start.bat         # Запуск локального сервера на Windows
```

## Локальный запуск (для разработки)

```bash
# Способ 1: Node.js
node server.js
# Открыть http://localhost:3000

# Способ 2: Python (если установлен)
python -m http.server 8000
# Открыть http://localhost:8000

# Способ 3: VS Code — установите расширение Live Server, кликните правой кнопкой на index.html → Open with Live Server
```
